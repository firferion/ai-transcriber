# AI Transcriber

📦 **[Скачать готовую программу (Transcriber_Monolith.exe)](https://github.com/firferion/ai-transcriber/releases/latest)**

AI Transcriber — это удобная программа с графическим интерфейсом (на базе Fyne) для локальной транскрибации видео и аудио в текст. 

Программа использует мощную модель нейросети **Whisper Large v3 Turbo** (через движок whisper.cpp) для высококачественного и быстрого распознавания речи (с фокусом на русский язык), а также **FFmpeg** для автоматического извлечения и подготовки аудиодорожек.

## Основные возможности:
* **Графический интерфейс**: Простой и понятный UI, не требующий работы в командной строке.
* **Поддержка форматов**: Работает с популярными видеоформатами (MP4, MKV, AVI, MOV, WEBM).
* **Аппаратное ускорение**: Поддержка обработки на видеокартах NVIDIA (GPU) для максимальной скорости, а также возможность работы на центральном процессоре (CPU).
* **Мониторинг прогресса**: Отображение процентов выполнения для этапов конвертации аудио и самой транскрибации.
* **Полная автономность**: После распаковки утилита работает полностью локально без необходимости подключения к интернету.

## Технологический стек
* **Язык**: Go (Golang)
* **GUI Framework**: Fyne
* **Движок**: whisper.cpp (модель ggml-large-v3-turbo-q5_0)
* **Обработка аудио**: FFmpeg

## Как использовать
1. Выберите видеофайл для обработки.
2. Укажите папку, куда будет сохранен итоговый текстовый файл.
3. Выберите устройство обработки (NVIDIA GPU или CPU).
4. Нажмите "ЗАПУСТИТЬ" и дождитесь окончания процесса. Текст будет сохранен в формате `.txt`.
